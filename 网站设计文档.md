## 一、整体设计概览

### 1.1 设计定位
极简暗黑风格的个人作品集网站，以黑色为主基调，橙色（#E06336）为强调色，通过 3D 卡片轮播 + Canvas 粒子文字特效构建沉浸式交互体验。

## 四、交互系统

### 4.1 3D 卡片轮播

**触发方式**:
| 操作 | 行为 |
|------|------|
| 左/右箭头点击 | `prevCard()` / `nextCard()` — 循环切换 |
| 导航栏链接点击 | `navigateTo(bookId)` — 直接跳转 |
| 轮播指示器点击 | `navigateTo(index)` — 直接跳转 |
| 键盘 ←/↑ | 上一张 |
| 键盘 →/↓ | 下一张 |
| 鼠标滚轮横向滚动 | 上一张/下一张（500ms 防抖） |
| 卡片 01 点击 | 打开个人简介覆盖层 |
| 卡片 02 点击 | 打开作品一览覆盖层 |
| 卡片 03 点击 | 打开联系方式覆盖层 |
| 卡片 04 点击 | 打开产品思考覆盖层 |

**实现逻辑**:
1. 维护 `currentIndex` 全局状态 (0~3)
2. 计算每张卡片与当前索引的距离 (distance = index - currentIndex)
3. 根据距离分配 CSS 类: `active`(0), `prev`(-1), `prev-2`(-2), `prev-3`(-3), `next`(+1), `next-2`(+2), `next-3`(+3)
4. 同步更新导航链接和指示器的 `active` 状态

**动画**: 0.6s cubic-bezier(0.4, 0, 0.2, 1)

### 4.2 覆盖层系统

**打开**: `overlay.classList.add('active')` + `document.body.style.overflow = 'hidden'`
**关闭**: `overlay.classList.remove('active')` + `document.body.style.overflow = ''`

**`navigateOverlay(bookId)` 函数**:
- 先关闭全部 4 个覆盖层
- 再根据 bookId 打开目标覆盖层 (0→profile, 1→portfolio, 2→contact, 3→productSense)
- 每个覆盖层内的导航链接均调用此函数实现跨界面跳转

**返回箭头**: 关闭当前覆盖层，返回主页

### 4.3 标签页切换

**个人简介界面** (4个标签): 关于我 / 教育背景 / 工作经历 / 生产工具
**作品一览界面** (5个标签): 情绪陪伴智能体 / 电商客服workflow / 合同审核 / 数字人 / 我的助理

**实现**: `switchProfileTab(tabId)` / `switchPortfolioTab(tabId)`
- 遍历标签添加/移除 `active` 类
- 遍历内容添加/移除 `active` 类
- 作品一览切换时内容滚动到顶部

### 4.4 图片灯箱

**打开**: `openLightbox(src, enableZoom)`
- 设置图片 src, 重置缩放(1)、位移(0,0)
- 若 `enableZoom=true`, 添加 `.zoomable` 类

**交互**:
- 滚轮缩放（仅 zoomEnabled）: 0.5× ~ 5× 范围, step ±0.1
- 拖拽平移（仅 scale>1 时）: mousedown → mousemove → mouseup
- 关闭: 未缩放时任意位置点击关闭; 缩放后仅点背景关闭

### 4.5 Canvas 粒子文字特效

**核心机制**:
1. 在目标区域用 Canvas 2D `fillText()` 或绘图 API 渲染文字/图案
2. 通过 `getImageData()` 采样像素（alpha > 150 = 有效）
3. 以 gap=4 步长遍历, 提取有效像素坐标作为粒子"目标位置"
4. 创建粒子数组, 粒子从 hero 区域外随机位置飞入
5. 每帧: `clearRect` → 更新所有粒子 → 绘制圆点

**粒子类 (Particle)**:

| 属性 | 值 | 说明 |
|------|-----|------|
| size | 1.5~4px 随机 | baseSize 备份用于恢复 |
| color | `hsl(0,0%,25%~90%)` | 随机灰度 |
| wanderAngle | 随机 0~2π | 游走方向 |
| wanderSpeed | 0.005~0.025 | 游走角速度 |
| wanderRadius | 0.5~2.5 | 游走幅度 |
| dying | false | 标记死亡粒子 |

**运动物理**:
- 常态 ease: 0.03（缓慢聚合）
- 过渡 ease: 0.12（快速重组）
- 速度阻尼: `vx/vy *= 0.88`
- 最大速度: 10px/帧
- 死亡动画: `size *= 0.9`, < 0.3 时移除

**鼠标交互**:
- 影响半径: 70px
- 排斥力: `(1 - distance/70) * 3`
- 靠近鼠标时粒子膨胀到 2x baseSize
- 触摸事件同样支持 (touchmove/touchend)

**文字切换 (`switchWord`)**:
- 4 个文字循环: `'Hi'` → `"I'm Hannah"` → `'SMILEY'` → `'Have a nice day'`
- 两阶段过渡:
  1. 散开阶段 (0~280ms): 粒子散射到随机方向
  2. 重组阶段 (280~780ms): 粒子飞向新文字的像素位置
  3. 完成后 `isTransitioning = false` 解锁
- `isTransitioning` 锁: 防止过渡期间重复触发
- 空目标保护: 若 `getTextPositions` 返回空数组, 中止切换并回退 index

**SMILEY 特殊渲染**: 使用 Canvas 绘图 API（`arc` 圆形脸 + 两个实心眼 + 弧线嘴）

**性能优化 — 位置缓存**:
- `positionCache`: 按文字内容索引的采样缓存
- 缓存项: `{ offW, offH, relPositions: [{rx, ry}, ...] }`
- 缓存命中条件: 宽度差 < 30px（容错滚动条）且高度差 < 5px
- 初始化时: 预渲染全部 4 个文字到缓存
- resize 时: 清空缓存并重新渲染全部文字
- scroll 时: 保留缓存, 仅用相对坐标重新计算绝对位置
- 最大粒子数: 6000（超出时均匀采样）

**Canvas 穿透点击**: 
- hero 区域内点击: 触发 `switchWord()`
- hero 区域外点击: 临时设置 `pointerEvents='none'`, 用 `elementFromPoint` 找到下方元素并转发 click 事件

## 七、关键数据流

```
用户交互
  │
  ├─ 点击导航链接 ──→ navigateTo(bookId) / navigateOverlay(bookId)
  │
  ├─ 点击卡片 ──→ (01→profile) (02→portfolio) (03→contact) (04→productSense)
  │
  ├─ 点击箭头 ──→ prevCard() / nextCard() → updateCarousel()
  │
  ├─ 键盘/滚轮 ──→ prevCard() / nextCard()
  │
  ├─ 点击指示器 ──→ navigateTo(index)
  │
  ├─ 点击粒子区域 ──→ switchWord()
  │                    ├── getTextPositions(word) → [缓存命中 / 渲染采样]
  │                    ├── 散射阶段 (0~280ms)
  │                    └── 重组阶段 (280~780ms)
  │
  ├─ window resize ──→ recalculatePositions(clearCache=true)
  │                    └── 清缓存 → 重渲染4个文字 → 更新粒子目标
  │
  └─ window scroll ──→ recalculatePositions(clearCache=false)
                       └── 保留缓存 → 仅重新计算绝对坐标
```

## 八、技术亮点

1. **单文件架构**: 全部 HTML/CSS/JS 内联，零外部依赖，部署简单
2. **CSS 3D Carousel**: `preserve-3d` + `translateZ` + `rotateY` 组合实现真实纵深
3. **Canvas 粒子系统**: 像素采样→粒子动画完整闭环，含智能缓存机制
4. **毛玻璃质感**: 多层 `backdrop-filter: blur()` 应用
5. **覆盖层管理**: 统一 open/close/navigate 逻辑 + body scroll lock
6. **图片拖拽缩放**: 灯箱支持 0.5×~5× 缩放 + 拖拽平移
7. **多模态输入**: 鼠标点击、键盘方向键、鼠标滚轮均可控制轮播
8. **触摸支持**: Canvas 粒子触控交互，覆盖层滚动隐藏滚动条
9. **Canvas 穿透点击**: 粒子画布不阻挡下层元素交互
10. **字体就绪保障**: `document.fonts.ready` + `requestAnimationFrame` 确保自定义字体加载后再初始化
